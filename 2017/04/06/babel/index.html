Babel的操作指南

Babel是一个通用的多用途JavaScript编译器

npm install babel-cli -g

把ES6编译成 ES5

安装依赖的预设

$ npm install --save-dev babel-preset-es2015

修改.babelrc

{
    "presets": ["es2015"]
}

$ babel es6.js  这将把编译后的结果直接输出至终端

$ babel es6.js -o es5.js  加上-o参数 可以将结果写入到指定的文件

$ babel src -d lib 把一个目录整个编译成一个新的目录
